Quick and easy way of toggling showing PHP errors. Very useful when developing!

== Description ==

This quick and small plugin makes it very easy to toggle seeing PHP errors right from the admin dashboard. Button only shows for administrators.

WARNING: The errors are sitewide and visible to anyone visiting the site, so it's only for development purposes.


**Instructions**

After you've installed and activated the plugin you'll find a button in the top WordPress admin bar allowing you to toggle showing PHP errors on and off. Simply press that and you're good to go.

== Installation ==

This section describes how to install the plugin and get it working. Recommended way of installation is by searching for the plugin in the 'Plugins' menu in WordPress. You can also follow the instructions below:

1. Download the zip-file containing the plugin and extract it.
2. Upload the `php-errors-toggle` directory to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Done!
